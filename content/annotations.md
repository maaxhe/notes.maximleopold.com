# Alle Annotationen

Diese Seite zeigt alle öffentlichen Annotationen auf dieser Website.

[→ Zur Annotations-Übersicht](/annotations)

## Was sind Annotationen?

Mit Hypothesis kannst du auf jeder Seite dieser Website Text markieren und kommentieren. Alle öffentlichen Annotationen werden hier gesammelt angezeigt.

### Wie erstelle ich eine Annotation?

1. Markiere beliebigen Text auf einer Seite
2. Klicke auf "Annotate" im Hypothesis-Popup
3. Schreibe deinen Kommentar
4. Wähle "Public" und klicke auf "Post"

Deine Annotation erscheint dann automatisch in dieser Übersicht!
